# PilotObjectInfo
Модуль позволяет посмотреть информацию о объектах Pilot:
-  атрибуты
-  дочерние объекты
-  создателя
-  статус
-  файлы с возможностью сохранения, добавления, удаления
-  права доступа
-  связи

Команда доступна через контекстное меню "Информации об объекте". Доступна для документов, объектов на Storage и задач.

## Релизы

Проект использует GitHub Actions для автоматической сборки и публикации релизов.

### Как создать релиз:

1. Создайте тег с версией (например, `v1.0.0`):
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```

2. GitHub Action автоматически:
   - Соберет проект в конфигурации Release
   - Упакует все файлы из `bin\Release\` в архив `PilotObjectInfo.ext2.zip`
   - Создаст GitHub Release с этим архивом

3. Скачайте собранный архив из раздела [Releases](../../releases)
